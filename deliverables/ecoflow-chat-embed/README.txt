Paquete de integración de chat de ecoFlow

Archivos incluidos:
- index.html
- style.css
- chat.js
- avatar.png
- ecoScanCalidad.png

Nota de integración:
- El frontend publica los mensajes de chat en /api/ecoflow/chat
- Si ecoSoftWEB sirve la barra lateral desde otra ruta base o dominio, adapte la URL de obtención en chat.js según corresponda.

- Este paquete contiene la capa de interfaz de usuario utilizada por el chat de ecoFlow para que su equipo pueda integrarlo o adaptarlo en una barra lateral
